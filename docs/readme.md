# Documentação do Projecto Orders Master Infoprex

Bem-vindo à documentação do projecto Orders Master Infoprex. Esta documentação descreve os padrões de desenho, arquitectura e regras estabelecidas para o desenvolvimento do projecto, com base nas definições do PRD v2.

De forma simples e directa, e garantindo que apenas é documentado o que já existe conceptualmente como regra estrutural (uma vez que o projecto inicia de um ambiente limpo), este repositório serve como base de conhecimento para onboarding de developers e manutenção.
